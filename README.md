# Christopher Gama - Professional Portfolio

## Water Resources Management & Development Specialist

This is the professional portfolio website of Christopher Gama, showcasing expertise in:
- Water Resources Management
- GIS & Remote Sensing
- Hydrological Modeling (SWAT)
- WASH Projects
- Data Analysis & MEAL Systems

## Features
- Responsive design (mobile, tablet, desktop)
- Interactive portfolio with filtering
- Data visualizations using Chart.js
- Interactive maps using Leaflet.js
- Dark/Light mode toggle
- Contact form with validation
- Animated skill bars
- Performance optimized

## Technologies Used
- HTML5, CSS3, JavaScript (ES6+)
- Chart.js for data visualizations
- Leaflet.js for interactive maps
- Font Awesome icons
- Google Fonts (Inter, Montserrat)

## Setup Instructions

1. **Clone or download** this repository
2. **Open `index.html`** in your web browser
3. **For local development:**
   - Use a local server (e.g., Live Server extension in VSCode)
   - All files are ready to use
