Spatio-Temporal Drought Analysis

Project Overview

This conceptual project demonstrates the use of Google Earth Engine and QGIS for analyzing drought patterns in the Lake Malawi Basin over a 10-year period.

Methodology

  • Collection of MODIS and Landsat satellite imagery
  • Calculation of NDVI and NDWI indices for vegetation and water stress
  • Time-series analysis of precipitation data (CHIRPS)
  • Spatial interpolation and vulnerability mapping

Tools & Technologies

Google Earth Engine QGIS Python JavaScript Remote Sensing
